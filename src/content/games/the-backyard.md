## About This Game

The Backyard is a charming isometric resource management game that captures the nostalgia of childhood outdoor play. Set in and around a kid's treehouse, players manage resources while engaging in classic backyard games.

### Features

- **Isometric View**: Beautiful isometric art style
- **Resource Management**: Gather and manage various resources
- **Childhood Theme**: Features classic outdoor games and activities
- **Treehouse Base**: Central hub for all operations

### Development

Created during a 48-hour GameJamPlus event with a team of 5 people.

### Technical Details

- Built with Unity and C#
- Custom isometric rendering system
- Resource management systems
- Event-driven game mechanics
