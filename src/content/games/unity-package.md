## About This Package

A comprehensive Unity package designed to accelerate project setup and provide commonly needed utilities for game development.

### Included Tools

- **Scene Changer**: Easy scene management and transitions
- **Project Folders Setup**: Automated folder structure creation
- **Advanced JSON Serialization**: Enhanced data persistence
- **And many more utilities**

### Purpose

This package was created to streamline the game development workflow by providing reusable tools that I commonly need across different projects.

### Features

- Easy installation via Unity Package Manager
- Modular design - use only what you need
- Well-documented API
- Regular updates and improvements
