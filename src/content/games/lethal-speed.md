## About This Game

Lethal Speed is a fast-paced action game inspired by Ghostrunner, focusing on speedrunning mechanics and fluid combat.

### Features

- Speedrunning-focused gameplay
- Advanced weapon system
- Dynamic character animations
- Stylized visual effects using Shader Graph

### My Role

As **Project Manager** of the 6-person team, I was responsible for:

- Leading the development team
- Implementing the weapon system
- Character animation integration
- Core gameplay systems
- Visual effects and polish

### Technical Highlights

- **Hierarchical State Machine AI**: Engineered advanced enemy AI using a hierarchical state machine approach
- **NavMesh Pathfinding**: Integrated Unity's NavMesh for intelligent enemy navigation
- **Custom Shaders**: Created visual effects using Shader Graph
