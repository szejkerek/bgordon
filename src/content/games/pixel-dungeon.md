## About This Game

Pixel Dungeon Escape is a challenging roguelike game where you must navigate through procedurally generated dungeons filled with monsters, traps, and treasures.

### Features

- Procedurally generated levels
- Permadeath mechanics
- Multiple character classes
- Pixel art graphics
- Original soundtrack

### Development

This game was created for GMTK Game Jam 2024 in just 48 hours. The theme was "Built to Scale" and I interpreted it as dungeons that scale in difficulty.

### Controls

- Arrow Keys: Move
- Space: Attack
- E: Interact
- I: Inventory
