## About This Game

Pirate Rush VR is an arcade VR game developed as my engineering thesis project. Inspired by Fruit Ninja, the game challenges players to slice or shoot down incoming projectiles in an immersive virtual reality environment.

### Features

- Engaging and smooth VR gameplay
- Dynamic object destruction system
- Dual interaction modes: slicing and shooting
- Immersive pirate-themed environment
- Developed for Oculus headsets

### Technical Highlights

- Built with Unity and Oculus XR SDK
- Custom physics-based slicing mechanics
- Optimized for VR performance
- Dynamic object pooling for smooth gameplay

### Development

This solo project served as my engineering thesis, demonstrating proficiency in VR development and game design principles.
